Python API
===============================================

the tools contained in **enstools-compression** can be accessed through the Python API.

Firstly, we provide a method :meth:`enstools.compression.api.compress` to compress a file or a list of files.

Secondly, the methods :meth:`enstools.compression.api.analyze_dataset` and :meth:`enstools.compression.api.analyze_files` to find appropriate compression specifications.
