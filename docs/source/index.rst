Welcome to Enstools-Compression's documentation! |docsbadge|
============================================================
This package extends `Ensemble Tools <https://enstools.readthedocs.io>`_ to enable **lossy** and **lossless** compression
by using **HDF5 filters**.

Also, it contains a set of tools to help to find appropriate compression specifications.
These tools can be accessed through a **Command Line Interface**.

.. |docsbadge| image:: https://readthedocs.org/projects/enstools-compression/badge/?version=latest
    :target: https://enstools-compression.readthedocs.io/en/latest/?badge=latest
    :alt: Documentation Status


It has been developed within the framework of **Waves to Weather** [1]_.
